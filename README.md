# Angular Todo Application

## Requirements
* PosgreSQL https://www.postgresql.org/download/

## Installation

Ruby version **2.3.3**

Install bundler gem

```console
#!bash

$ gem install bundler
```

Install dependencies

```console
#!bash

$ bundle install
```

Database creation

```console
#!bash
$ rake db:create
$ rake db:migrate

```

Run Rails Server

```
#!bash
$ rails server
```

Access in browser [http://localhost:3000](http://localhost:3000)
